Proyecto integrador para las materias Gestión de Proyectos de Software y Sistemas Programables

El robot se conecta a la app via bluetooth y luego es capaz de conectarse a una red wifi, mientras mantiene un servidor el monitor web puede acceder a información del robot.
Además los robots pueden enviarse mensajes entre sí con antenas RF.
El robot cuenta con un modo de control manual y autónomo, así como sensores de proximidad y detector de linea para este último modo.
El código del robot también hace uso de una pantalla OLED y un buzzer para transmitir información al usuario.
